## 📌 Main Features
- Kakao 로그인을 지원
- 판매글을 작성하고 사진을 업로드 할 수 있습니다. 게시글에 좋아요를 눌러줄 수 있습니다.
- 채팅방에 새로운 채팅이 있을 경우 SSE를 이용한 실시간 알람이 발송됩니다.
- 작성된 판매글을 실시간 채팅을 이용한 문의가 가능하고, 채팅을 상대가 읽었는지 여부를 확인할 수 있는 읽지 않음 표시 기능이 제공됩니다.
- 동네 선택, 카테고리 별 등록된 판매글을 필터링 할 수 있습니다.
